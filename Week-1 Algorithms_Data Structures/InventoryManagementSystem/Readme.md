# Inventory Management System

## 1. Why Data Structures and Algorithms are Important

Data structures and algorithms help efficiently manage large inventories by
improving the speed of searching, inserting, updating, and deleting products.

## 2. Suitable Data Structures

- ArrayList
- HashMap

HashMap is preferred because it provides faster lookup using Product ID.

## 3. Time Complexity

| Operation | Time Complexity |
|------------|-----------------|
| Add | O(1) |
| Update | O(1) |
| Delete | O(1) |
| Search | O(1) |
| Display | O(n) |

## 4. Optimization

Using HashMap reduces the time required for searching, updating, and deleting products.
